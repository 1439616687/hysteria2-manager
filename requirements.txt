# ==================== Version Notes ====================
# 
# Version Compatibility:
# - All packages are pinned to specific versions for stability
# - Compatible with Python 3.6+
# - Tested on Ubuntu 20.04/22.04, Debian 11/12, CentOS 8/9
# 
# Security Updates:
# - All dependencies use latest stable versions as of 2024
# - No known security vulnerabilities in listed versions
# 
# Performance Considerations:
# - Core dependencies are lightweight (~15MB total)
# - Optional dependencies can be added as needed
# - Production dependencies add ~30MB
# 
# Installation Notes:
# 1. Create virtual environment:
#    python3 -m venv venv
# 
# 2. Activate virtual environment:
#    source venv/bin/activate  # Linux/Mac
#    venv\Scripts\activate     # Windows
# 
# 3. Install dependencies:
#    pip install -r requirements.txt
# 
# 4. For development:
#    pip install -r requirements-dev.txt
# 
# 5. For production:
#    pip install -r requirements-prod.txt
# 
# Upgrade Notes:
# - To upgrade all packages:
#   pip install --upgrade -r requirements.txt
# 
# - To check for updates:
#   pip list --outdated
# 
# - To generate updated requirements:
#   pip freeze > requirements.txt
# 
# Troubleshooting:
# - If bcrypt fails to install:
#   apt-get install python3-dev libffi-dev  # Debian/Ubuntu
#   yum install python3-devel libffi-devel  # CentOS/RHEL
# 
# - If PyYAML fails to install:
#   apt-get install libyaml-dev  # Debian/Ubuntu
#   yum install libyaml-devel    # CentOS/RHEL
# 
# - If psutil fails to install:
#   apt-get install gcc python3-dev  # Debian/Ubuntu
#   yum install gcc python3-devel    # CentOS/RHEL
